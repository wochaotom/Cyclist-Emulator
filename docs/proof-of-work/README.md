# Proof of work

Consultation 1 explicitly asks for "ancillary" materials — handwritten derivations, trial-and-error notes, initial code drafts — that show a **documented history of group discussion and revision**. A model that appears with no such history gets scrutinized.

Drop that material here:

- Photos/scans of handwritten derivations and scratch work
- Meeting notes and decisions
- Whiteboard snapshots

**Naming:** `YYYY-MM-DD-short-description.jpg` (e.g. `2026-06-03-drag-derivation.jpg`).

The git commit history is the other half of this evidence — commit your own work, from your own account, as you go.
