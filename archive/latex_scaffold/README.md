# Archived LaTeX scaffold

`main.tex` is a LaTeX skeleton for the write-up, kept for reference only. It has been updated to describe the **canonical** time-stepping model in `notebooks/model_v3.ipynb` — the force balance, the gradient-driven power target, the power-threshold fatigue mechanic, and the rider × course results table.

It is **not** the team's write-up. The actual paper is a shared document that follows the section outline in `docs/paper-outline.md` and is exported to PDF — no LaTeX required. This file is unmaintained and kept only as a structural reference.

The prose sections and the wind / power-deviation / validation results are left as `% TODO` (they need runs on the notebook). The figure includes were removed, so it is a text-only skeleton that compiles on its own.
